# HR Management System

## Setup Instructions

### Backend Setup
1. cd backend
2. npm install
3. Update .env with your database credentials
4. Import database/schema.sql into MySQL
5. npm start

### Frontend Setup
1. cd frontend
2. npm install
3. Update .env with backend URL
4. npm run dev

Default Admin Login:
- Username: admin
- Password: admin123

## Features
- Admin Authentication
- Job Vacancy Management
- Application Management
- Employee Management
- Attendance Management with automatic weekend calculation
